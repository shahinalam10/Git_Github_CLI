Working-directory git command

mkdir folderName

Ex: mkdir myFolder

This command is used for making a new folder

cd folderName

Ex: cd myFolder

This command is used changing directory

ls -a

Ex: just type ls -a

This command is all file/folder include hidden

touch filename.exte

Ex: touch index.php

This command is used to a file create from project folder

git --version

Ex: git --version

This command is used to check the git version

git config --global user.name

Ex: git config --global user.name 'Shahin Alam'

This command is used define the username of the git user globally

git config --global user.email

Ex: git config --global user.email 'shahin@email.com'

This command is used define the email of the git user globally

git init

Ex: git init

The git init command creates a new Git repository. It can be used to convert an existing,project to a Git initialize

git status

Ex: git status

This command is used to check status for track and untrack file


Git is the version control system that manages the changes in code, while GitHub is a platform that hosts Git repositories and provides additional collaboration features for developers
